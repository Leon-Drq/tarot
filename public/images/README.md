# Tarot Card Image Sources

The local card images `22.png` through `35.png` are the Wands suit front images.
They were derived from Wikimedia Commons files `Wands01.jpg` through `Wands14.jpg`
from the Rider-Waite-Smith tarot deck, illustrated by Pamela Colman Smith.
Source category: https://commons.wikimedia.org/wiki/Category:Rider-Waite-Smith_tarot_deck_(TaionWC)

The images were downloaded on 2026-06-30 and resized to 300x527 PNG files to
match the app's existing tarot card image dimensions.
